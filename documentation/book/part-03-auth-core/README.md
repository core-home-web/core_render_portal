# Part 3: Authentication & Core Systems

This section covers the authentication system, user management, API routes, and React context providers that form the core infrastructure of the application.

## Chapters in This Part

| Chapter | Title | What You'll Learn |
|---------|-------|-------------------|
| [Chapter 11](./chapter-11-authentication.md) | Authentication System | Supabase Auth integration, login/signup flows |
| [Chapter 12](./chapter-12-user-management.md) | User Management | User profiles, team assignment, settings |

## Learning Objectives

After completing Part 3, you will have:

1. A fully working authentication system with login/signup
2. User profile management with team-based theming
3. API routes for email sending and file uploads
4. Global state management via React Context

## Prerequisites

Before starting Part 3:

- Completed Parts 1 and 2 (project setup and database)
- Supabase project with Auth enabled
- Understanding of React hooks and context
