# Part 1: Foundation & Setup

This section covers the fundamentals of the Core Render Portal, from understanding what it does to setting up your development environment.

## Learning Objectives

After completing Part 1, you will:

1. Understand the purpose and business value of the Render Portal
2. Have all necessary tools and accounts set up
3. Have a working Next.js project with the correct structure
4. Have a configured Supabase project connected to your application
5. Understand how all parts of the system work together

## Prerequisites

Before starting Part 1, ensure you have:

- Basic knowledge of JavaScript/TypeScript
- Familiarity with React concepts
- A computer with Node.js installed (or ability to install it)
- Internet access for downloading dependencies
