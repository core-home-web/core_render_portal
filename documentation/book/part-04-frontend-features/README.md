# Part 4: Frontend Features

This section covers building the main application features including project management, multi-step forms, image annotation, whiteboard, collaboration, and export functionality.

## Chapters in This Part

| Chapter | Title | What You'll Learn |
|---------|-------|-------------------|
| [Chapter 15](./chapter-15-project-management.md) | Project Management | Creating, editing, viewing, and deleting projects |
| [Chapter 16](./chapter-16-multi-step-forms.md) | Multi-Step Forms | Form wizard implementation with validation |
| [Chapter 17](./chapter-17-image-annotation.md) | Image Annotation | Canvas system and part markers |
| [Chapter 18](./chapter-18-whiteboard.md) | Whiteboard Integration | Excalidraw setup and collaboration |
| [Chapter 19](./chapter-19-collaboration.md) | Collaboration System | Invitations, permissions, sharing |
| [Chapter 20](./chapter-20-export.md) | Export Functionality | PNG, SVG, JSON, HTML exports |

## Learning Objectives

After completing Part 4, you will have:

1. Complete project CRUD functionality
2. Multi-step form wizard with validation
3. Image annotation with canvas overlays
4. Collaborative whiteboard with auto-save
5. Invitation and permission system
6. Multiple export formats

## Prerequisites

Before starting Part 4:

- Completed Parts 1-3 (setup, database, auth)
- Understanding of React hooks and state
- Familiarity with form handling

## Feature Dependencies

