# Prerequisites

Before setting up the Core Render Portal, you need to install and configure the following tools.

## Required Software

### 1. Node.js (v18 or higher)

```bash
node --version  # Should output v18.x.x or higher
```

**Install:** [nodejs.org](https://nodejs.org) or `brew install node`

### 2. pnpm (Recommended)

```bash
npm install -g pnpm
pnpm --version  # Should output 8.x.x or higher
```

### 3. Git

```bash
git --version
```

### 4. Code Editor

Recommended: VS Code with extensions:
- ESLint
- Prettier
- Tailwind CSS IntelliSense

## Required Accounts

### 1. Supabase Account

1. Go to [supabase.com](https://supabase.com)
2. Create an account
3. Create a new project
4. Save your database password

### 2. GitHub Account (Optional)

For repository access and deployment.

### 3. Vercel Account (For Deployment)

Sign up at [vercel.com](https://vercel.com) with GitHub.

## Checklist

- [ ] Node.js v18+ installed
- [ ] pnpm installed
- [ ] Git installed
- [ ] Code editor (VS Code)
- [ ] Supabase account and project created
