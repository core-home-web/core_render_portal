# First Run

Start the development server and verify everything works.

## Start the Server

```bash
pnpm dev
```

You should see:
```
▲ Next.js 14.0.3
- Local: http://localhost:3000
✓ Ready
```

## Access the App

1. Open **http://localhost:3000**
2. You should see the login page

## Create Your First Account

1. Click **"Sign Up"**
2. Enter email and password
3. Select your team
4. Click **"Create Account"**

## Create Your First Project

1. Click **"New Project"**
2. Fill in project details
3. Add items and parts
4. Click **"Create Project"**

## Test Whiteboard

1. Open a project
2. Click **"Whiteboard"**
3. Draw something
4. Verify it saves

## Available Pages

| URL | Page |
|-----|------|
| `/` | Home |
| `/auth/login` | Login |
| `/auth/signup` | Signup |
| `/dashboard` | Dashboard |
| `/projects` | All projects |
| `/project/new` | Create project |
| `/project/[id]` | View project |
| `/project/[id]/whiteboard` | Whiteboard |
| `/settings` | Settings |

## Congratulations!

You've successfully set up the Core Render Portal!
